# Setup CI/CD Command

One-command CI/CD pipeline generation.

## Usage

```bash
/setup-ci-cd
```

Generates:
- GitHub Actions workflows
- Chromatic visual regression
- Vercel/Netlify deployment
- PR preview comments
- Bundle size tracking

Interactive setup guides you through configuration.
